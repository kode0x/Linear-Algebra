# Linear Algebra (Visual Notes)

A collection of **concise linear algebra notes** combined with **interactive visualizations and animations** using Python. This project is built in **Jupyter Notebook** to make abstract concepts intuitive and visually engaging.

This project aims to simplify core concepts of linear algebra by combining:

- **Short, structured notes**
- **Visual representations**
- **Animations for deeper intuition**

Instead of only focusing on theory, the notebooks emphasize **understanding through visualization**, making it ideal for students and self-learners.
